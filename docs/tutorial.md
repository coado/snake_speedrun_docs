# Tutorial

W pierwszej kolejności użytkownik wybiera kolor węża, mapy i trudność gry (łatwa, średnia, trudna). Następnie po kliknięciu "Graj" rozpoczyna się rozgrywka. Podczas gry użytkownik zdobywa punkty za każdą sekundę gry oraz za każdy item, który zje jego wąż (ma znaczenie szybkość wzrostu do czasu gry - im szybciej wąż zjada tym więcej punktów dostaje, więc ważne, żeby nie pomijać itemków). Następnie dane są zbierane przez system i zapisywane w bazie danych. W panelu głównym przed każdą rozgrywką wyświetlane są statyski gracza z poprzednich gier.
